# aicw-website-builder

Website builder API server that builds and deploys static websites to Cloudflare Pages.

## Setup

Clone with submodules:

```bash
git clone --recursive git@github.com:emirn/aicw-website-builder.git
```

Or if already cloned:

```bash
git submodule update --init --recursive
```

## Development

See `api-server/README.md` for API documentation and development instructions.
