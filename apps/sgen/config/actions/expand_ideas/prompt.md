## Instructions

For each idea above, create one or more article proposals:

1. **Expand each idea** into a concrete article proposal with:
   - An SEO-optimized title (50-65 characters)
   - A URL-friendly path (lowercase, hyphenated, no trailing slash)
   - 3-5 target keywords (primary + related)
   - A clear description (max 155 characters) that serves as both a writing brief and meta description

2. **Split broad ideas** into 2-3 focused articles when the idea covers too much ground for a single piece.

3. **Merge overlapping ideas** if two ideas would produce very similar articles.

4. **Match the website's niche** — tailor titles, keywords, and angles to fit the website context above.

5. **Vary content types** — mix how-to guides, comparisons, lists, and in-depth explainers where appropriate.
